树莓派快速启动filebrowser文件浏览器
https://github.com/filebrowser/filebrowser/
之前是caddyserver的一个组件，后来独立出来的。
```
docker run -d 
    --name=filebrowser 
    --restart unless-stopped 
    -v /mnt/:/mnt/pi/ 
    --publish 8099:80
    --env UID=1000
    --env GID=1000 
    --env GIDLIST=1000 
    filebrowser/filebrowser
```

这样的话就可以访问
访问地址：ip:8099
帐号密码：admin/admin
进入里面设置下目录到/mnt下就可以了
