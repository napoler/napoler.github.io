## 美杜莎

“\`

# 下载美杜莎镜象

docker pull linuxserver/medusa

# 运行实例

docker create  
 –name=medusa  
 –restart unless-stopped  
 -v /home/pi/data/medusa:/config  
 -v /home/pi/dirve/:/media/AiDisk\_a1  
 -e PGID=1000 -e PUID=1000 

 -e TZ=Asia/Shanghai  
 -p 8081:8081  
 linuxserver/medusa

“\`  
这样子你的美杜莎就执行在 ip:8081上了
