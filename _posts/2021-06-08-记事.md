## PyTorch One Hot 编码

PyTorch has a `one_hot()` function for converting class indices to one-hot encoded targets:

```python
import torch
import torch.nn.functional as F

x = torch.tensor([4, 3, 2, 1, 0])
F.one_hot(x, num_classes=6)
```
